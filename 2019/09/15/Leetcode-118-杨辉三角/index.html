杨辉三角

题目描述

给定一个非负整数 numRows，生成杨辉三角的前 numRows 行。

在杨辉三角中，每个数是它左上方和右上方的数的和。

示例:

输入: 5
输出:
[
[1],
[1,1],
[1,2,1],
[1,3,3,1],
[1,4,6,4,1]
]

思路

创建一个二维数组，第 n 行有 n 个元素，对每一行的第一个元素赋值 1 ，其他的元素的值为arr[i] [j] = arr[i - 1] [j - 1] + arr[i - 1] [j]
